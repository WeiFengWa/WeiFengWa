## API

| 参数      | 说明     | 类型                    | 默认值       |
| --------- | -------- | ----------------------- | ------------ |
| size      | 间距大小 | `number`                | `8`          |
| direction | 间距方向 | `horizontal` `vertical` | `horizontal` |
| wrap      | 是否换行 | `boolean`               | `false`      |
